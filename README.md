# results/

This folder contains output plots generated by the bonus task notebook.

## What gets saved here

| File | Description |
|------|-------------|
| `reconstruction_comparison.png` | Original vs reconstructed for all 4 latent sizes |
| `loss_curves.png` | Training loss for each latent dimension experiment |
| `tsne_latent_space.png` | t-SNE visualization of latent space (colour = class) |
| `denoising_results.png` | Noisy input vs denoised reconstruction |

> These files are generated when you run the bonus task notebook.
> Run all cells and the plots save here automatically.
